## 🚀 시작하기

### 접속 방법

1. 웹 브라우저에서 애플리케이션 주소로 접속합니다.
2. 처음 사용하는 경우 회원가입을 진행합니다.
3. 이미 계정이 있는 경우 로그인합니다.

---

## 👤 회원가입 및 로그인

### 회원가입

1. 메인 페이지에서 **"회원가입"** 버튼을 클릭합니다.
2. 다음 정보를 입력합니다:
    - 이메일
    - 비밀번호
    - 이름
    - 핸드폰번호
    - 생년월일
    - 포지션 (GK, DF, MF, FW)
3. **"회원가입"** 버튼을 클릭하여 가입을 완료합니다.

### 로그인

1. 메인 페이지에서 **"로그인"** 버튼을 클릭합니다.
2. 이메일과 비밀번호를 입력합니다.
3. **"로그인"** 버튼을 클릭합니다.

### 카카오 로그인

1. 로그인 페이지에서 **"카카오 로그인"** 버튼을 클릭합니다.
2. 카카오 계정으로 로그인합니다.

---

## ⚙️ 프로필 설정

### 첫 로그인 시 프로필 설정

1. 첫 로그인 시 **"기본 회원정보를 작성해주세요"** 페이지로 이동합니다.
2. 다음 정보를 입력합니다:
    - 핸드폰번호
    - 생년월일
    - 포지션
    - 프로필 사진 (선택)
    - 소개글 (선택)
3. 정보 입력 후 팀 선택 페이지로 이동합니다.

### 프로필 수정

1. 대시보드에서 **"마이페이지"** 탭을 클릭합니다.
2. **"프로필 수정"** 버튼을 클릭합니다.
3. 수정할 정보를 변경합니다:
    - 이름
    - 핸드폰번호
    - 생년월일
    - 포지션
    - 프로필 사진
    - 소개글
4. **"저장하기"** 버튼을 클릭합니다.

### 비밀번호 변경

1. 대시보드에서 **"마이페이지"** 탭을 클릭합니다.
2. **"비밀번호 변경"** 버튼을 클릭합니다.
3. 현재 비밀번호와 새 비밀번호를 입력합니다.
4. **"변경하기"** 버튼을 클릭합니다.

---

## 👥 팀 생성 및 가입

### 팀 생성

1. 로그인 후 **"클럽 선택"** 페이지로 이동합니다.
2. **"클럽 생성"** 카드를 클릭합니다.
3. 다음 정보를 입력합니다:
    - 클럽 이름
    - 지역
    - 클럽 소개 (선택)
4. **"생성하기"** 버튼을 클릭합니다.
5. 팀 생성 시 자동으로 팀장이 됩니다.

### 팀 가입 신청

1. **"클럽 선택"** 페이지에서 **"클럽 가입신청"** 카드를 클릭합니다.
2. 가입하고 싶은 팀을 검색하거나 목록에서 선택합니다.
3. **"팀정보"** 버튼을 클릭하여 팀 상세 정보를 확인합니다.
4. **"가입신청"** 버튼을 클릭합니다.
5. 팀장의 승인을 기다립니다.

### 가입 신청 관리 (팀장/부팀장)

1. 대시보드에서 **"마이팀"** 탭을 클릭합니다.
2. **"가입승인관리"** 탭을 클릭합니다.
3. 가입 신청 목록을 확인합니다.
4. 각 신청에 대해 **"수락"** 또는 **"거절"** 버튼을 클릭합니다.

### 가입 신청 취소

1. **"클럽 선택"** 페이지 상단에 가입 신청 대기 중인 팀이 표시됩니다.
2. **"신청 취소"** 버튼을 클릭하여 신청을 취소할 수 있습니다.

---

## 🏆 경기 관리

### 경기 등록 (팀장/부팀장)

1. 대시보드에서 **"경기"** 탭을 클릭합니다.
2. **"경기 등록"** 버튼을 클릭합니다.
3. 다음 정보를 입력합니다:
    - 날짜 (필수)
    - 시간 (선택)
    - 장소 (선택)
    - 상대팀 이름 (필수)
4. **"등록하기"** 버튼을 클릭합니다.

### 경기 수정 (팀장/부팀장)

1. 경기 목록에서 수정할 경기를 클릭합니다.
2. 경기 상세 페이지에서 **"수정"** 버튼을 클릭합니다.
3. 정보를 수정한 후 **"수정하기"** 버튼을 클릭합니다.

### 경기 삭제 (팀장/부팀장)

1. 경기 상세 페이지에서 **"삭제"** 버튼을 클릭합니다.
2. 확인 메시지에서 **"확인"**을 클릭합니다.

### 경기 목록 조회

1. 대시보드에서 **"경기"** 탭을 클릭합니다.
2. 년도와 월을 선택하여 필터링할 수 있습니다.
3. **"전체 조회"**를 선택하면 모든 경기를 볼 수 있습니다.

---

## ✅ 참석 투표

### 투표하기

1. 대시보드 **"요약"** 탭에서 다음 경기 정보를 확인합니다.
2. **"투표하러가기"** 버튼을 클릭합니다.
3. 경기 상세 페이지에서 다음 중 하나를 선택합니다:
    - **"참석"**: 경기에 참석합니다.
    - **"미정"**: 참석 여부를 아직 결정하지 않았습니다.
    - **"불참"**: 경기에 참석하지 않습니다.
4. 선택한 버튼을 클릭하면 투표가 완료됩니다.

### 투표 현황 확인

1. 경기 상세 페이지에서 **"참석 현황"** 섹션을 확인합니다.
2. 팀원들의 참석/불참/미정 상태를 확인할 수 있습니다.

---

## 📊 경기 기록 입력 (팀장/부팀장)

### 기록 입력하기

1. 경기 상세 페이지에서 **"기록 입력"** 버튼을 클릭합니다.
2. 다음 정보를 입력합니다:

    **경기 점수 입력:**

    - 각 게임별 우리팀 점수와 상대팀 점수를 입력합니다.
    - **"게임 추가"** 버튼으로 여러 게임을 추가할 수 있습니다.

    **선수 기록 입력:**

    - 각 선수별로 출전 여부를 체크박스로 선택합니다.
    - 출전한 선수의 경우 **"골|도움"** 버튼을 클릭하여 득점과 도움을 입력합니다.
    - 각 게임별로 선수 기록을 입력할 수 있습니다.

    **회고 메모 (선택):**

    - 경기 후기를 작성할 수 있습니다.

3. **"저장하기"** 버튼을 클릭하여 기록을 저장합니다.

### 기록 확인

1. 경기 상세 페이지에서 **"게임별 상세"** 섹션을 확인합니다.
2. 각 게임의 점수와 선수 기록을 확인할 수 있습니다.

## 👥 팀원 관리

### 팀원 추가 (팀장)

1. 대시보드에서 **"팀 구성"** 탭을 클릭합니다.
2. **"회원관리"** 탭을 클릭합니다.
3. **"팀원 추가"** 버튼을 클릭합니다.
4. 사용자 ID를 입력하고 등번호를 설정합니다.
5. **"추가하기"** 버튼을 클릭합니다.

### 부팀장 임명 (팀장)

1. **"회원관리"** 탭에서 부팀장으로 임명할 팀원을 찾습니다.
2. **"부팀장 임명"** 버튼을 클릭합니다.
3. 부팀장은 최대 2명까지 임명할 수 있습니다.

### 등번호 설정/수정 (팀장)

1. **"회원관리"** 탭에서 등번호를 설정할 팀원을 찾습니다.
2. 등번호 필드를 클릭하여 숫자를 입력합니다.
3. 중복된 등번호는 사용할 수 없습니다.

### 팀원 상태 변경 (팀장)

1. **"회원관리"** 탭에서 상태를 변경할 팀원을 찾습니다.
2. 상태 드롭다운에서 다음 중 하나를 선택합니다:
    - 활동
    - 부상
    - 휴식
    - 기타

### 팀원 추방 (팀장)

1. **"회원관리"** 탭에서 추방할 팀원을 찾습니다.
2. **"팀 추방"** 버튼을 클릭합니다.
3. 확인 메시지에서 **"확인"**을 클릭합니다.

### 팀 탈퇴 (일반 팀원)

1. 대시보드에서 **"마이팀"** 탭을 클릭합니다.
2. **"팀 탈퇴"** 버튼을 클릭합니다.
3. 확인 메시지에서 **"확인"**을 클릭합니다.

### 팀 삭제 (팀장)

1. 대시보드에서 **"마이팀"** 탭을 클릭합니다.
2. **"팀 삭제"** 버튼을 클릭합니다.
3. 확인 메시지에 **"팀 삭제"**를 입력하고 **"확인"**을 클릭합니다.

## 📈 통계 및 랭킹

### 요약 탭

대시보드 **"요약"** 탭에서 다음 정보를 확인할 수 있습니다:

-   **다음 경기**: 다음 경기 정보와 내 참석 상태
-   **팀 구성**: 총 인원수와 포지션별 인원수
-   **Top10**: 출전수, 득점, 도움, 승률 상위 10명
-   **팀 통계**: 경기 통계, 게임 통계, 총 득점
-   **참석 현황**: 경기별 참석 현황

### 랭킹 탭

1. 대시보드에서 **"랭킹"** 탭을 클릭합니다.
2. 다음 탭에서 랭킹을 확인할 수 있습니다:
    - **출전수**: 경기 출전 횟수 순위
    - **득점**: 총 득점 순위
    - **도움**: 총 도움 순위
    - **승률**: 승률 순위
    - **포인트**: 종합 포인트 순위

### Top10 조회

1. **"요약"** 탭의 **"Top10"** 카드에서 버튼을 클릭합니다:
    - **"출전수"**: 출전 횟수 상위 10명
    - **"득점"**: 득점 상위 10명
    - **"도움"**: 도움 상위 10명
    - **"승률"**: 승률 상위 10명

## 👤 마이페이지

### 프로필 확인

1. 대시보드에서 **"마이페이지"** 탭을 클릭합니다.
2. 다음 정보를 확인할 수 있습니다:
    - 프로필 사진
    - 이름
    - 소속 팀 정보
    - 프로필 정보 (이메일, 핸드폰번호, 생년월일, 포지션)
    - 소개글

### 프로필 수정

1. **"마이페이지"**에서 **"프로필 수정"** 버튼을 클릭합니다.
2. 수정할 정보를 변경합니다.
3. **"저장하기"** 버튼을 클릭합니다.

## 💡 유용한 팁

1. **참석 투표**: 경기 전에 반드시 참석 투표를 완료하세요. 팀장이 참석 인원을 파악하는 데 도움이 됩니다.

2. **경기 기록**: 경기 후 가능한 한 빨리 경기 기록을 입력하세요. 통계와 랭킹이 정확하게 반영됩니다.

3. **등번호**: 등번호는 팀 내에서 중복될 수 없습니다. 팀장이 등번호를 설정할 때 주의하세요.

4. **부팀장**: 부팀장은 최대 2명까지 임명할 수 있습니다. 팀장이 부재 시 부팀장이 팀을 관리할 수 있습니다.

5. **가입 신청**: 팀에 가입하고 싶을 때는 가입 신청을 보내세요. 팀장이 승인하면 자동으로 팀원이 됩니다.

6. **검색 기능**: 팀 선택 페이지에서 팀 이름, 지역, 소개로 검색할 수 있습니다.

## ❓ 문제 해결

### 로그인이 안 될 때

-   이메일과 비밀번호를 다시 확인하세요.
-   비밀번호를 잊어버린 경우 비밀번호 변경 기능을 사용하세요.

### 팀에 가입할 수 없을 때

-   이미 다른 팀에 소속되어 있는지 확인하세요.
-   가입 신청이 대기 중인지 확인하세요.

### 경기 기록을 입력할 수 없을 때

-   팀장 또는 부팀장 권한이 있는지 확인하세요.
-   경기가 이미 종료되었는지 확인하세요.

### 투표가 반영되지 않을 때

-   페이지를 새로고침해보세요.
-   대시보드 요약 탭으로 돌아가면 자동으로 업데이트됩니다.
